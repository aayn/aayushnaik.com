- Forbes 2026 World's Billionaires List (https://www.forbes.com/billionaires/) — primary source; the gated interactive list. Forbes website blocks direct fetch; data routed through Forbes-mirroring API above.
- Reference for prior rank band 1501-2000: `research/billionaires/chunk_05.md` (same methodology, same API).

### Coverage
- Total rows: 500 of 500 expected (455 distinct ranks; 500 distinct individuals)
- Net worth filled: 500/500 (100%)
- Country filled: 499/500 (99%)
- Source of wealth filled: 500/500 (100%)
- Self-Made Score: 30 from Forbes direct, 348 inferred (range only), 122 unknown
- Inherited Y/N: 343/500 derived (68%); 157 unknown

### Gaps and caveats
- **Self-Made Score is the largest gap.** The Forbes API exposes the precise 1-10 `selfMadeRank` for only 30 of 500 records at this depth. For the rest, only a boolean `selfMade` flag is available. I list "unknown (likely 6-10)" when the boolean is True, and "unknown (likely 1-5)" when the name carries "& family" (a strong inheritance marker in Forbes' own listing convention). 122 rows have no signal at all and are marked plain "unknown".
- **Tied ranks**: Forbes assigns the same rank to people of identical archived net worth (often family co-heirs). I keep all 500 individuals listed; some share a rank slot — this matches Forbes' published convention.
- **Net worth precision**: I report `archivedWorth` (the published 2026-list snapshot, in $M, divided by 1000 to give $B). This is the figure Forbes published on March 10, 2026; live `finalWorth` will differ.
- **"& family" entries** are listed by Forbes as a single dynasty unit. The name reflects multiple living relatives sharing a stake; the score logic flags these as likely-inherited or mixed depending on the boolean `selfMade` flag.
- **Source description granularity**: Forbes' `source` field is short (e.g. "Investments", "Real estate"). I append the broader `industries` taxonomy for context.
- I did not enrich beyond the Forbes API (e.g. by individual Wikipedia lookup). At 500 rows, that would be a disproportionate effort and would risk fabrication — the API data is itself sourced from Forbes' published 2026 list.
