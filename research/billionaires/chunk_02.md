## Notes
- **Primary source:** Forbes 2026 World's Billionaires List via mirror API `https://forbes400.onrender.com/api/forbes400` (snapshot 2026-05-09; aligns with Forbes published list dated March 10, 2026).
- **Cross-check:** Forbes profile pages on `forbes.com/profile/<uri>/` (mirror retains the same `uri` slug), and Forbes-direct realtime API `forbesapi/person/rtb/0/finalWorth/desc.json` (used by chunk 05).
- **Coverage:** 300 of 300 rows filled. 100% on rank, name, net worth, country, source of wealth, industry classification.
- **Self-Made Score:** 120/300 rows have a numeric Forbes `selfMadeRank` (Forbes assigns these only to U.S.-based billionaires; non-U.S. profiles return `null`).
- **Inherited:** 231/300 rows have a Y/N derived from either the numeric score (1–5 = Y, 6–10 = N) or the Forbes `selfMade` boolean. Where both are absent and there is no "& family" hint, the cell reads `unknown`.
- **Tie ranks:** Forbes assigns identical ranks to billionaires of equal net worth (e.g. four people share rank 221). All such ties are preserved as-is; the table contains 300 rows even though only 284 rank values appear.
- **Net worth precision:** Values come from `finalWorth` in $M, divided by 1,000 and shown to one decimal in $B. Real-time worths can drift slightly from the printed list.
- **Caveats:**
  - The Forbes `selfMade` boolean is binary (true/false) — "likely 6–10" / "likely 1–5" qualifiers reflect that resolution, not a fabricated score.
  - `& family` naming is Forbes' convention for collective family fortunes; treated as ambiguous (`unknown`) since the family group itself can include both founders and heirs.
  - A handful of profiles use a primary-residence country that differs from citizenship; this table uses `countryOfCitizenship` for consistency with chunk 05.
