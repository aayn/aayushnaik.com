## Notes

### Sources used
- Forbes 400 unofficial API (https://forbes400.onrender.com/api/forbes400) — mirrors Forbes real-time billionaires list, sourced from Forbes; data captured 2026-05-09. Provides rank, name, net worth (`finalWorth`/`archivedWorth`), citizenship, source description, industries, `selfMade` boolean, and (for some) `selfMadeRank` (1-10).
- Forbes 2026 World's Billionaires List (https://www.forbes.com/billionaires/) — primary source; the gated interactive list. Forbes website blocks direct fetch; data routed through Forbes-mirroring API above.
- Wikipedia (https://en.wikipedia.org/wiki/The_World%27s_Billionaires) — top-10 only at this depth; not useful for ranks 1501-2000.
- Caproasia summary (https://www.caproasia.com/2026/03/26/forbes-top-billionaires-list-2026...) — top-line stats only.

### Coverage
- Total rows: 500 of 500 expected (some rank slots tied; 468 distinct ranks; 500 distinct individuals)
- Net worth filled: 500/500 (100%)
- Country filled: 500/500 (100%)
- Source of wealth filled: 500/500 (100%)
- Self-Made Score: 58 from Forbes direct, 300 inferred (range only), 142 unknown
- Inherited Y/N: 313/500 derived (62%); 187 unknown

### Gaps and caveats
- **Self-Made Score is the largest gap.** The Forbes API exposes the precise 1-10 `selfMadeRank` for only 58 of 500 records at this depth. For the rest, only a boolean `selfMade` flag is available. I list "unknown (likely 6-10)" when the boolean is True, and "unknown (likely 1-5)" when the name carries "& family" (a strong inheritance marker in Forbes' own listing convention). 142 rows have no signal at all and are marked plain "unknown".
- **Tied ranks**: Forbes assigns the same rank to people of identical archived net worth (often family co-heirs, e.g. Mars siblings, Wertheimer brothers). I keep all 500 individuals listed; some share a rank slot — this matches Forbes' published convention.
- **Net worth precision**: I report `archivedWorth` (the published 2026-list snapshot, in $M, divided by 1000 to give $B). This is the figure Forbes published on March 10, 2026; live `finalWorth` will differ.
- **"& family" entries** (53 in this chunk) are listed by Forbes as a single dynasty unit (e.g. "Stefan Quandt & family"). The name reflects multiple living relatives sharing a stake; the score logic flags these as likely-inherited.
- **Source description granularity**: Forbes' `source` field is short (e.g. "Pharmaceuticals", "Real estate"). I append the broader `industries` taxonomy for context.
- I did not enrich beyond the Forbes API (e.g. by individual Wikipedia lookup). At 500 rows, that would be a disproportionate effort and would risk fabrication — the API data is itself sourced from Forbes' published 2026 list.
