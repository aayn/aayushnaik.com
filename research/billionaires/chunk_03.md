### Caveats
- **Net worth scale**: API returns `finalWorth` in millions USD; values divided by 1,000 to produce $B. The companion `archivedWorth` (the original Forbes 2026 published figure as of March 10, 2026) is similar but generally lower since `finalWorth` reflects daily real-time updates. Values shown here use `finalWorth` (most current as of capture date 2026-05-09), which may differ slightly from the static March 2026 published list.
- **Tied ranks**: Forbes assigns the same rank to billionaires with identical net worth; this slice contains 447 distinct rank slots across 500 individuals (e.g., rank 505 has two co-founders of Canva).
- **Self-Made Score**: Only available directly from Forbes for 164 of 500 individuals (~33%). For the rest, the `selfMade` boolean was used as a proxy: `selfMade=true` → likely 6–10 (or 4–6 if name contains "& family"); `selfMade=false` → likely 1–5. The "& family" heuristic flags entries where wealth is shared/transitioned through a family unit (Forbes typically scores these 4–6).
- **Inherited Y/N**: Per task spec, Y if Forbes self-made score is 1–5 (substantial inheritance/structural advantage), N if 6–10 (substantially self-made). When score is inferred from boolean, Y/N follows the inference; when "& family" makes the inference ambiguous, "unknown" is used.
- **Source description vs Industry**: "Source of Wealth" combines Forbes' free-text `source` field (specific business/asset) with `industries[]` array (sector taxonomy) for richer context.
- **Total net worth in this slice**: ~$2875.0B across ranks 501–1000.
